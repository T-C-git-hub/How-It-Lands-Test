# How-It-Lands-Test
How-It-Lands Test Repo

The "How-It-Lands" Test predicts how: website pricing / landing pages; research paper abstracts, or grant summaries will land when released publicly.

The test uses two cohorts: low-attention readers and high attention readers. Low-attention readers mimic skim readers under first-contact. High-attention readers mimic author-familiar readers.

The test measures (across both cohorts): perceived category of the specimen, perceived intent of the specimen, trust-trigger points, trust-level, shareabiliy of specimen.

Input: above-the fold web-page or (up-to) 3 paragraphs of text.

Output: single-pass, no changes prescribed, see [How-It-Landed (sample output).pdf](https://github.com/T-C-git-hub/How-It-Lands-Test/blob/main/How-It-Landed%20(sample%20output).pdf)
